## What is Advancius?

The Advancius network is a cross-platform Minecraft server made by [Monkeyboys](https://namemc.com/monkeyboys).
The server was created on January 2nd, 2018. The goal of the Network is to bring the best experience a Minecraft server can provide, with our amazing community, dedicated staff team, and continuous changes to make sure the network is always at its best.

The Advancius network is unique from othe servers because it offers the ability to earn **free ranks** by playing games within the network, earning in-game currency, and more!

## Gamemodes

The Advancius network offers a wide range of different gamemodes to play, such as Skyblock, Factions, Prison, etc.
If you would like more information about each individual gamemode on the network, please visit the [gamemodes page](/category/gamemodes).

## Ranks

Ranks are products that people can earn or buy on the Advancius Network server. Once you have your rank either earned by accumulating enough of our in-game currency, donating, or by being accepted for a staff position, you have the ability to access exclusive features with each rank.
If you would like more information about the perks for each rank, please visit the [ranks page](/category/ranks).

## Approved Mods

Please be aware that all use of modifications on the Advancius network is at your own risk. Using any modifications or macros that provide a clear advantage to in-game players is never allowed.
We cannot review every version of every modification, and, as such, we cannot guarantee that all versions of the modifications fall within our server rules.

Please also note that if you are downloading modifications, you should always download them from the official source of that modification. Unofficial downloads may result in the modifications not performing as expected and may result in punishment. To get the list of approved mods allowed on the Advancius network, please visit the [approved mods list](https://www.advancius.net/approvedmods).
